# py-talk

同じWi-Fi（ローカルネットワーク）内にある2台のPC（Mac / Windows）間で、テキストファイルを直接送受信するためのシンプルなPythonスクリプトです。

外部のクラウドサービスやチャットツールを使わず、Pythonの標準ライブラリのみで高速かつ安全に通信します。

## 特徴
- **環境構築不要**: 追加のライブラリ（`pip` 等）のインストールは一切必要ありません。
- **アカウント不要**: 会員登録やログインの手間がありません。
- **ローカル完結**: データがインターネットを経由しないため安全です。

## 使い方

### 1. 受け取り側の準備
ファイルを受け取りたいPCで `receive.py` を実行します。

```bash
python receive.py
```

実行すると、画面に以下のようにIPアドレスが表示されます。
> `受け取り待機中... このPCのIPアドレスは: 192.168.x.x`

このIPアドレスをメモしてください。

### 2. 送り側の準備と送信
ファイルを送りたいPCで `send.py` を開き、以下の2箇所を書き換えます。

```python
SERVER_IP = "192.168.x.x"  # 1でメモした受け取り側のIPアドレスを入力
FILE_PATH = "test.txt"     # 送りたいテキストファイルのパスを入力
```

修正後、スクリプトを実行します。

```bash
python send.py
```

> `送信が完了しました！` と表示されれば転送完了です。受け取り側と同じフォルダーにファイルが保存されます。

## 注意事項
- 2台のPCが**同じWi-Fi（ネットワーク）**に接続されている必要があります。
- 接続できない場合は、セキュリティソフトやOSのファイアウォール設定でポート `50001` がブロックされていないか確認してください。
